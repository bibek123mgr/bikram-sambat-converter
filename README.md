## 📖 Overview

**Nepali Date Converter** is a powerful, zero-dependency JavaScript library for converting dates between the **Gregorian (AD)** calendar and the **Bikram Sambat (BS)** calendar, which is the official calendar of Nepal. The library provides full support for Nepali language outputs including Nepali digits (२०८१) and Nepali month names (बैशाख).

### ✨ Features

- 🔄 **Bi-directional Conversion** - AD ↔ BS
- 📅 **Wide Date Range** - BS 2000-2089 (AD 1943-2033)
- 🔢 **Nepali Digit Support** - Automatically converts to Nepali numbers
- 📝 **Multi-language Output** - English and Nepali month/day names
- ✅ **Built-in Validation** - Handles invalid dates gracefully
- 🎯 **Zero Dependencies** - Lightweight and fast
- 🌐 **Cross-platform** - Works in Node.js and browsers
- 📦 **ES6+ Support** - Modern JavaScript with static methods
